Hva er JavaScript?

  • Skapt av Brendan Eich som del av nettleseren Netscape i 1995
  • Standardisert som ECMAScript
  • Utviklingen skjer gjennom mange løp, med mange aktører involvert

Syntaks

Variabler

var a; var b = 42; var c = "forty two";

Funksjoner

function sumA (a, b) { return a + b; } var sumB = function (a, b) { return a + b; }; var sumC = new Function("a", "b", "return a + b;");

Kallemåter:

sumA(4, 5); sumA.call({}, 4, 5); sumA.apply({}, [4, 5]);

Objekter

var objectA = { a: 42 }; var objectB = new Object(); objectB.a = 42; var constructor = new function (a) { this.a = a; }; var objectC = new constructor(42);

Lister

var arrayA = [1, 2]; var arrayB = new Array(1, 2); var arrayC = []; arrayC.push(2); arrayC.unshift(1);
